# Linux & Networking Home Lab

## Overview

This repository documents my journey learning Linux system administration and computer networking while preparing for the CompTIA Network+ certification.

The labs are performed using:

- Raspberry Pi Zero W
- Raspberry Pi OS
- Windows 11
- Ubuntu Server (planned)
- Wireshark
- PowerShell

Each lab includes:

- Objectives
- Network diagrams
- Configuration steps
- Commands used
- Verification
- Troubleshooting
- Lessons learned
